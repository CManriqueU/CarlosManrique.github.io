Tydi - Programming sample

In this project I used the Core Motion framework, more specifically the accelerometer and gyroscope, the app simply prints the values for the x,y,z coordinates that both the accelerometer and gyroscope read simultaneously, that can be viewed only using a physical phone as simulator. I also added Alamofire to my project and used it with an API designed for testing, once you hit the 'start a request' button a table will be shown as text in a UITextView box. The project is ready for further development and more features can be added, such as: adjusting the elements in the Main.storyboard and displaying more data. also, handling other types of responses in the app and work with different parameters,adding constraints, and creating an app icon.
