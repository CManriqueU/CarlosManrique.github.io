Blog with AWS Lambda + AuroraDB + Rest API

This project leverages AWS services, including Lambda, REST API, and AuroraDB. Its primary goal is to enable CRUD operations within the comment section of the blog "A night in Tokyo and New york". Users will be able to view existing comments on the blog (READ), post new comments (CREATE), which will be stored in the database, update existing comments (UPDATE), and finally, delete comments from the blog, which will also remove them from the database.
